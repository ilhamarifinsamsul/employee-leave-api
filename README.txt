1) REQUIREMENT SISTEM
A. Backend
- PHP >= 8.2
- Composer
- Laravel 12.x
- MySQL
- XAMPP (Untuk local server)
- Postman (Untuk test API)
B. Frontend
- Blade Template (Laravel)

2) CARA INSTALLASI
----------------------------------------
1. Clone / Extract project
   - Jika dari Git:
     git clone <repo_url>
   - Jika dari zip/rar:
     Extract ke folder project

3) CARA MENJALANKAN APLIKASI
----------------------------------------
1. Jalankan server Laravel
   php artisan serve

2. Akses aplikasi:
   http://127.0.0.1:8000

3. Akses endpoint API:
   http://127.0.0.1:8001/api/leave-requests

4) INFORMASI AKUN LOGIN (Jika ada)
----------------------------------------
Aplikasi ini tidak menggunakan login / autentikasi.
